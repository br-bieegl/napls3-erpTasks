PS3 BD Remote Control maping to Microsoft Remote Control

  1. Hold "ENTER"+"START" buttons of the Remote and click "Query Remote" button to pair.
  2. Edit maping value("FF" will be no function!) . refer to MCE RC-6 Code Table
  3. Click "Enable" button. the maping value you have edited will be loaded automatically next time.
